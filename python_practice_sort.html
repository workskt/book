ソート#

リストのソート#


リストの中身を並び替えたい場合は、以下の二通りの方法があります。
(1) sort()を使う
(2) sorted()を使う

(1) sort()を使う

これはリスト型のメソッドですので、ソートしたいリストとsort()を.で繋ぎます。
指定したリストの要素が並び替えられ、上書きされます。

a = ['TP53', 'EGFR', 'MYC', 'CDKN1A']
a.sort()
print(a)

['CDKN1A', 'EGFR', 'MYC', 'TP53']

リストaの要素が、アルファベット順に並び替えられました。
上書きされるので、元の順番を持つ配列は残っていません。

(2) sorted()を使う

これは組み込み関数（最初から使える関数）で、新しいリストを返します。

組み込み関数の一覧はこちらから。
https://docs.python.org/ja/3/library/functions.html
組み込み関数でないものは、事前にimportする必要があります（後述）。

a = ['TP53', 'EGFR', 'MYC', 'CDKN1A']
b = sorted(a)
print(b)
print(a)

['CDKN1A', 'EGFR', 'MYC', 'TP53']
['TP53', 'EGFR', 'MYC', 'CDKN1A']

新しく定義したbに並び替えられたリストが代入されました。
元のリストaはそのまま残っています。

降順に並べ替えたい場合は、カッコ内にreverse=Trueと書きます。

a = ['TP53', 'EGFR', 'MYC', 'CDKN1A']
a.sort(reverse=True)
print(a)

['TP53', 'MYC', 'EGFR', 'CDKN1A']

a = ['TP53', 'EGFR', 'MYC', 'CDKN1A']
b = sorted(a, reverse=True)
print(b)

['TP53', 'MYC', 'EGFR', 'CDKN1A']

文字列のソート#


文字列を並び替える場合は、sorted()を使用します。
結果はリスト型として返されます。

a = 'tumor protein p53'
b = sorted(a)
print(b)
print(type(b))
c = sorted(a, reverse=True)
print(c)
print(type(c))

[' ', ' ', '3', '5', 'e', 'i', 'm', 'n', 'o', 'o', 'p', 'p', 'r', 'r', 't', 't', 'u']
<class 'list'>
['u', 't', 't', 'r', 'r', 'p', 'p', 'o', 'o', 'n', 'm', 'i', 'e', '5', '3', ' ', ' ']
<class 'list'>

リスト型だと使いにくいので、文字列に戻す場合は、リスト型のメソッドjoin()を使います。
join()はリストの要素を、指定した区切り文字をはさんで連結してくれます。


'区切り文字'.join()


d = ''.join(b)
print(d)
print(type(d))

  35eimnoopprrttu
<class 'str'>

カンマで区切って出力することもできます。

d = ','.join(b)
print(d)

 , ,3,5,e,i,m,n,o,o,p,p,r,r,t,t,u

辞書のソート#


辞書を並び替えたい場合も、sorted()を使用します。

キーを並び替えたい場合は、keys()を使います。
結果はリスト形式で返されます。

genes_dict = {'TP53': 100, 'EGFR': 300, 'MYC': 400, 'CDKN1A': 200}
a = sorted(genes_dict.keys())
print(a)
print(type(a))

['CDKN1A', 'EGFR', 'MYC', 'TP53']
<class 'list'>

要素を並び替えたい場合は、values()を使います。

genes_dict = {'TP53': 100, 'EGFR': 300, 'MYC': 400, 'CDKN1A': 200}
b = sorted(genes_dict.values())
print(b)

[100, 200, 300, 400]

降順で並べ替えたい場合は、カッコ内にreverse=Trueと書きます。

genes_dict = {'TP53': 100, 'EGFR': 300, 'MYC': 400, 'CDKN1A': 200}
c = sorted(genes_dict.values(), reverse=True)
print(c)

[400, 300, 200, 100]

さて、要素の順番で並び替えて、その順番でキーを取得したいときはどうすればよいでしょうか。
遺伝子名と発現量が格納された辞書を使って、遺伝子を発現量順に並べるといった作業が必要になることはよくあります。

この場合、辞書型のメソッドitems()、key関数、lambda関数を使用します。
結果はタプル型で返されます。

genes_dict = {'TP53': 100, 'EGFR': 300, 'MYC': 400, 'CDKN1A': 200}
d = sorted(genes_dict.items(), key=lambda x:x[1], reverse=True)
print(d)

[('MYC', 400), ('EGFR', 300), ('CDKN1A', 200), ('TP53', 100)]

要素の降順に並び替えられて、タプル型で結果が返ってきました。

sorted()のカッコ内がちょっとややこしいですが、まずは理屈抜きで丸暗記して使えるようになりましょう。

もし遺伝子名をリストで取得したければ、リスト内包表記を使いましょう。

genes_dict = {'TP53': 100, 'EGFR': 300, 'MYC': 400, 'CDKN1A': 200}
e = [gene for gene, expression in sorted(genes_dict.items(), key=lambda x:x[1], reverse=True)]
print(e)

['MYC', 'EGFR', 'CDKN1A', 'TP53']

練習#

(1)上記のgenes_dictの辞書に格納された遺伝子を、発現量が大きい順に並び替えて出力するコードを作成してください。
(2)上記のgenes_dictの辞書に格納された遺伝子のうち、発現量が300以上かどうか判定し、300以上だった場合はその遺伝子名と発現量を出力するコードを作成してください。
