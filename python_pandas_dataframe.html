pandasデータフレームの扱い方¶

テストデータの取得¶


（すでにタブ区切りのテストデータが手元にある場合は、下の「データの読み込み」まで読み飛ばしてください。）


データフレームを扱うために、pandasをインポートします。

また、定番中の定番データであるirisを取得するため、load_irisをインポートします。

import pandas as pd
from sklearn.datasets import load_iris
iris = load_iris()

irisはdictionary-like objectのようですので、とりあえずkeys()で項目を確認してみます。

iris.keys()

dict_keys(['data', 'target', 'frame', 'target_names', 'DESCR', 'feature_names', 'filename'])

データの中身も確認してみます。

sklearn.datasetsのデータを確認するためには、データの変数と項目名をドットで繋ぎます。

iris.feature_names

['sepal length (cm)',
 'sepal width (cm)',
 'petal length (cm)',
 'petal width (cm)']

‘data’は各個体のデータ、’target’は品種のインデックス、’target_names’は品種名、’feature_names’はデータ項目が格納されているようです。

扱いやすくするために、pandasのDataFrame()でデータフレームの形式に変換します。

columnsで列ラベル情報を指定します。

df = pd.DataFrame(iris.data, columns=iris.feature_names)

データフレームのファイルへの出力¶


このデータフレーム形式をいったんタブ区切りファイルに出力します。

出力のためにはto_csv()を使います。

()内に出力ファイルのパスを記入します。

sep='\t'を指定することでタブ区切りファイルとして出力できます。

指定しない場合はcsvファイルとして出力します。

行ラベル(index)は必要ないので、index=Falseを指定します。

もし列ラベルが不要であれば、header=Falseを指定します。

df.to_csv('irisdata.txt', sep='\t', index=False)
